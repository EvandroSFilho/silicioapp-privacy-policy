Política de Privacidade – SilicioApp

Última atualização: Março de 2026

1. Informações Coletadas

O SilicioApp pode coletar as seguintes informações fornecidas pelo usuário:

  • Nome e sobrenome
  • Endereço de e-mail
  • Dados de produção inseridos no aplicativo
  • Informações necessárias para autenticação

2. Como Utilizamos as Informações

As informações coletadas são utilizadas exclusivamente para:

  • Criar e gerenciar a conta do usuário
  • Permitir login e autenticação
  • Armazenar dados de produção
  • Gerar relatórios e arquivos em PDF
  • Melhorar a experiência do aplicativo

3. Autenticação e Serviços de Terceiros

O SilicioApp utiliza o serviço de autenticação fornecido pelo Firebase Authentication, uma plataforma desenvolvida pelo Google.

4. Armazenamento de Dados

Os dados de produção inseridos pelo usuário são armazenados localmente no dispositivo por meio de banco de dados interno do aplicativo.

5. Compartilhamento de Dados

O SilicioApp não compartilha dados pessoais com terceiros, exceto quando necessário para:

  • Cumprimento de obrigação legal
  • Funcionamento do serviço de autenticação

6. Segurança

O SilicioApp adota medidas técnicas para proteger os dados dos usuários.

7. Direitos do Usuário

Nos termos da LGPD, o usuário pode:

  • Solicitar a exclusão de sua conta
  • Atualizar seus dados
  • Solicitar informações sobre os dados armazenados

8. Exclusão de Dados

O usuário pode excluir seus dados desinstalando o aplicativo.

9. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente.
