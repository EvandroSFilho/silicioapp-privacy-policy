Política de Privacidade – SilicioApp

Última atualização: Março de 2026

O SilicioApp respeita a sua privacidade e está comprometido em proteger os dados pessoais dos seus usuários, conforme a Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018).

1. Informações Coletadas

O SilicioApp pode coletar as seguintes informações fornecidas pelo usuário:

  • Nome e sobrenome
  • Endereço de e-mail
  • Dados de produção inseridos no aplicativo
  • Informações necessárias para autenticação

O aplicativo não coleta dados pessoais sensíveis.

2. Finalidade do Uso das Informações

As informações coletadas são utilizadas exclusivamente para:

  • Criar e gerenciar a conta do usuário
  • Permitir login e autenticação
  • Armazenar dados de produção
  • Gerar relatórios e arquivos em PDF
  • Garantir o funcionamento adequado do aplicativo

O SilicioApp não vende, aluga ou comercializa dados pessoais.

3. Serviços de Terceiros

O SilicioApp utiliza o serviço Firebase Authentication, fornecido pela Google, para autenticação e recuperação de senha.

O Firebase pode processar:

  • Endereço de e-mail
  • Credenciais de autenticação

Esses dados são utilizados exclusivamente para permitir o funcionamento do login e recuperação de senha do usuário.

Para mais informações, consulte a Política de Privacidade do Firebase.

4. Armazenamento de Dados

Os dados de produção são armazenados localmente no dispositivo do usuário por meio de banco de dados interno do aplicativo.

Os dados de autenticação são gerenciados pelo Firebase Authentication.

5. Compartilhamento de Dados

O SilicioApp não compartilha dados pessoais com terceiros, exceto quando necessário para:

  • Funcionamento do serviço de autenticação (Firebase)
  • Cumprimento de obrigação legal

6. Segurança

O SilicioApp adota medidas técnicas para proteger os dados dos usuários, incluindo:

  • Transmissão segura de dados via HTTPS/TLS (Firebase)
  • Criptografia de senhas
  • Assinatura digital do aplicativo
  • Proteção contra acesso não autorizado

7. Direitos do Usuário

Nos termos da LGPD, o usuário pode:

  • Solicitar a exclusão de sua conta
  • Atualizar seus dados
  • Solicitar informações sobre os dados armazenados

8. Exclusão de Dados

O usuário pode solicitar a exclusão permanente da conta e dos dados associados.

A exclusão pode ser solicitada:

  • Pelo próprio aplicativo (quando disponível)
  • Por solicitação via e-mail

Ao excluir a conta:

  • Os dados de produção armazenados localmente serão removidos
  • A conta e credenciais armazenadas no Firebase Authentication serão permanentemente excluídas

Solicitações são processadas em até 7 dias úteis.

9. Account Deletion (For Google Play Compliance)

Users may request permanent account deletion by sending an email to:

silicioapp.suporte@gmail.com

Subject: Account Deletion Request

Please include the email address used to register in the app.

Upon verification, the following data will be permanently deleted:

  • User name
  • Email address
  • Production records
  • Authentication data stored in Firebase

No personal data is retained after account deletion.

10. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
